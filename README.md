
# Webpack-Usage

This repository was created to understand and facilitate the use of webpack in projects.


## Run Locally

Clone the project

```bash
  git clone https://github.com/burakibaoglu0/Webpack-Usage
```

Go to the project directory

```bash
  cd Webpack-Usage
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build the project

```bash
    npm run build
```
